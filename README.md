# Life Tracker PWA v15

Änderungen:
- Verbrauch zeigt jetzt immer ursprüngliche Schätzung und berechneten Durchschnitt getrennt an.
- Termine zeigen jetzt ursprüngliches Intervall und berechnetes Intervall getrennt an.
- Erledigte Termine werden in einer Historie gespeichert, damit die App nach mehreren Daten ein eigenes Intervall berechnen kann.
- Überfällige Termine im Dashboard können direkt als „Heute erledigt“ oder „Erledigt am Datum“ abgehakt werden.
- Beim Abhaken wird automatisch das nächste Fälligkeitsdatum aus berechnetem Intervall oder ursprünglichem Intervall berechnet.
- Service Worker/Cache auf v15 gesetzt.
